The `Switch` component provides a toggle control for binary states, featuring a sliding animation when toggled. It's commonly used for enabling or disabling features and settings.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={<BasicSwitch />}
  codes={[
    {
      code:
`<Switch
    label="Enable notifications"
    description="Receive updates about new features and improvements"
    isChecked={isChecked}
    onToggle={() => setIsChecked(!isChecked)}
/>`,
      language: "tsx",
      label: "Switch"
    }
  ]}
/>

## Loading

Show a loading spinner while an action is being processed.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={<SwitchWithLoading />}
  codes={[
    {
      code:
`const [isChecked, setIsChecked] = useState(false);
const [isLoading, setIsLoading] = useState(false);

const handleToggle = () => {
    setIsLoading(true);
    
    // Simulate API call with timeout
    setTimeout(() => {
      setIsChecked(!isChecked);
      setIsLoading(false);
    }, 1000);
};

<Switch
    label="Processing"
    isChecked={isChecked}
    loading={isLoading}
    onToggle={handleToggle}
/>`,
      language: "tsx",
      label: "Loading"
    }
  ]}
/>
