Once UI utilizes a spacing system based on a custom 8px grid to provide options for setting white spaces effortlessly and consistently.

## Spacing system

Once UI relies on the type called `SpacingToken` to define static and responsive spaces. It's used for margins, paddings, gaps, and position distances. Static spaces are set in REMs and can be referenced with their pixel value calculated with a 16px root font size. For a more illustrative example, visit the [Structure](/once-ui/basics/structure) page. These spacings will slightly adjust based on your `--font-scaling` variables.

<Row gap="4" paddingTop="64" marginBottom="24">
  {[0, 1, 2, 4, 8, 12, 16, 20, 24, 32, 40, 48, 56, 64, 80, 104, 128, 160].map((size) => (
    <Background
      key={size}
      radius="s"
      overflow="visible"
      lines={{display: true, angle: -45, color: "neutral-alpha-weak", size: "8", thickness: 1}}
      background="neutral-alpha-weak"
      border="neutral-alpha-medium"
      center
      fillWidth
      style={{ 
        height: `var(--static-space-${size})`,
      }}
    >
      <InlineCode position="absolute" style={{ top: '-2rem' }}>{size}</InlineCode>
    </Background>
  ))}
</Row>

Responsive spaces are also set in REMs, but they are rescaled at the `m` and `s` breakpoints. They can be accessed through the T-shirt size model.

    <Background

        height: `var(--responsive-space-${size})`,
      }}
    >
      <InlineCode position="absolute" style={{ top: '-2rem' }}>{size}</InlineCode>
    </Background>
  ))}
</Row>
