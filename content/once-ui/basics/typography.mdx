Once UI manages typography properties with the [`Text`](/once-ui/components/text) and 
[`Heading`](/once-ui/components/heading) components. They apply utility classes and CSS variables under the hood. See the [Config](/once-ui/config) page for our recommended font setup.

## Font size

Add `font-size` classes to elements. The value adjusts to the adjacent or closest parent `font-family` class.

<Table
  marginTop="16"
  marginBottom="24"

      [".font-xl", "dynamic"],
      [".font-l", "dynamic"],

      [".font-s", "dynamic"],
      [".font-xs", "dynamic"]
    ]
  }}
/>
