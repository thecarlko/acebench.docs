Shape language in design profoundly influences the style of an interface. For instance, a website employing a large border radius often appears more playful, whereas one with small borders and sharp corners tends to convey a more conservative feel.

## Shape style

Override the default shape style with the `data-border` attribute on a native HTML element.

<CodeBlock
  label="Shape style"
  marginTop="16"
  marginBottom="24"
  codes={[
    {
      code: `<Flex data-border="conservative | playful | rounded" radius="m"/>`,
      language: "tsx",
      label: "Radius style"
    }
  ]}
  preview={
    <Flex fillWidth mobileDirection="column" gap="m">
      <Background
        data-border="conservative"
        background="neutral-alpha-weak"
        lines={{display: true, color: "neutral-alpha-weak"}}
        shadow="m"
        fillWidth
        radius="m"
        height={4}
        border="neutral-alpha-medium"
      />
      <Background
        data-border="playful"
        background="neutral-alpha-weak"
        lines={{display: true, color: "neutral-alpha-weak"}}
        shadow="m"
        fillWidth
        radius="m"
        height={4}
        border="neutral-alpha-medium"
      />
      <Background
        data-border="rounded"
        background="neutral-alpha-weak"
        lines={{display: true, color: "neutral-alpha-weak"}}

        radius="m"

  }
/>

## Nested radius

A common challenge in border-radius systems is nesting. For nested elements to maintain an aesthetic and natural appearance, they require varying border radius sizes. Once UI addresses this issue by providing additional border radius values specifically for containers with `static-size-4` and `static-size-8` paddings, which are the most frequently used nesting sizes.

<CodeBlock
  label="Nested radius"
  marginTop="16"
  marginBottom="24"
  codes={[
    {
      code:
`const radius = { xs | s | m | l | xl }

<Flex radius="{radius}-8">
    <Flex radius="{radius}-4">
      <Flex radius="{radius}"/>
    </Flex>
</Flex>`,
      language: "tsx",
      label: "Nested radius"
    }
  ]}
  preview={
    <Flex fillWidth mobileDirection="column" gap="m">

        <Flex fillWidth padding="4" border="neutral-strong" radius="xs-4">

            height={4}
            background="neutral-alpha-weak"

            shadow="m"
            border="neutral-alpha-medium"
