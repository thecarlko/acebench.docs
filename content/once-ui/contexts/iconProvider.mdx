The `IconProvider` component is a context provider that allows you to pass an object of icon instances to your application. Add any icon from the [react-icons](https://react-icons.github.io/react-icons/) library to the provider.

## Simple setup

Wrap your app with `IconProvider` and pass it the icons as props.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={24}
  lineNumbers
  codes={[
    {
      code:
`import { HiHome } from "react-icons/hi2";

<IconProvider icons={{home: HiHome}}>
    {children}
</IconProvider>`,

  ]}
/>

## Advanced setup

You can create a separate file for all icons, for example `icons.ts`. We also recommend using a `Providers` component to wrap your app.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codeHeight={32}
  lineNumbers
  codes={[
    {
      code:
`import { iconLibrary } from "../resources/icons.ts";
import { IconProvider, ToastProvider, ThemeProvider, DataThemeProvider } from "@once-ui-system/core";

export function Providers({ children }: {
    children: React.ReactNode
}) {
    return (
      <ThemeProvider>
        <DataThemeProvider>
          <ToastProvider>
            <IconProvider icons={iconLibrary}>
              {children}
            </IconProvider>
          </ToastProvider>
        </DataThemeProvider>
      </ThemeProvider>
    );
}`,
      language: "tsx",

## Default icons

Overwrite the default icons used in Once UI components (for example the chevron in the `Select` component) by providing new icons with the same keys to the `IconProvider`.

## Icon component

The <SmartLink suffixIcon="arrowUpRight" href="/once-ui/components/icon">`Icon`</SmartLink> component is a wrapper around icon instances to control their size and color. We recommend
inserting icons through this component.
