## Config

The default settings for the data viz module are stored in the `src/app/resources/data.config.js` file. You can use it to define the default height, highlight color, data mode and variant.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  codes={[
    {
      code:

mode: "categorical", // categorical | divergent | sequential
height: 24, // default chart height
axisLine: {
    stroke: "var(--neutral-alpha-weak)",
},
tick: {
    fill: "var(--neutral-on-background-weak)",
    fontSize: 11,
},
tickLine: false,`,

      label: "Config"
    }
  ]}

## Props

All charts share the same props for common settings, so it's easy to swap between different chart types with minimal code changes. Each chart may have additional, chart-specific props.

### Data

Use the `series` and `data` props to define the chart data to render, and assign colors to override defaults.

The `series` prop is an array of objects, where each object has a `key` property that defines the name of the series, and an optional `color` property that defines the color of the series. Match the `key` property with the property name in the `data` prop.

<BarChart
  marginTop="16"
  title="Daily Time Spent on Activities"
  axis="none"
  barWidth="xl"
  legend={{
    position: "bottom-center",
  }}
  series={[
    { key: "Reading", color: "aqua" },
    { key: "Sports", color: "yellow" },
    { key: "Doomscrolling", color: "orange" }
  ]}
  data={[
    { label: "Minutes per day", "Reading": 16, "Sports": 36, "Doomscrolling": 250 },
  ]}
/>
