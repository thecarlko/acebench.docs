The Logo component is used to display brand logos with consistent styling and sizing. It supports displaying both an icon and a wordmark, either separately or together.

<Feedback marginY="24" vertical="center" description={<Text>Replace the SVG files in the <InlineCode>public/trademark</InlineCode> folder with your own brand logo files. It will be used as the default logo and automatically render in the correct theme.</Text>}/>
