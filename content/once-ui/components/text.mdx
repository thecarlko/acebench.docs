The `Text` component provides consistent typography throughout your UI, with support for variants, color tokens, alignment, and spacing. It is typically used for body content, labels, descriptions, and captions.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column gap="12">
      <Text variant="body-default-m">This is default body text.</Text>
      <Text variant="label-strong-s">This is a strong label.</Text>
      <Text size="s" weight="strong">This is small, strong text.</Text>
    </Column>
  }
  codes={[
    {
      code:
`<Text variant="body-default-m">
  This is default body text.
</Text>
<Text variant="label-strong-s">
  This is a strong label.
</Text>
<Text size="s" weight="strong">
  This is small, strong text.
</Text>`,
      language: "tsx",
      label: "Text"
    }

/>

## Colors

Text color can be set using `onBackground` or `onSolid`.

  marginTop="16"
  marginBottom="24"

      </Row>
      <Row gap="12" fillWidth radius="l" padding="12" horizontal="center">
        <Text onBackground="accent-weak">Accent weak</Text>
      </Row>
      <Row gap="12" fillWidth radius="l" padding="12" horizontal="center">
        <Text onBackground="brand-weak">Brand weak</Text>
      </Row>

    <Row gap="12" fillWidth mobileDirection="column">

      <Row gap="12" solid="accent-medium" fillWidth radius="l" padding="12" horizontal="center">

        <Text onSolid="brand-weak">Brand medium</Text>
