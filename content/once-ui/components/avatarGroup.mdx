A component for displaying multiple user avatars in a compact, horizontal stack. Commonly used in collaborative interfaces to show team members, participants, or contributors.

## Limit

Use the `limit` prop to control how many avatars are displayed. Any additional avatars will be represented by a count indicator.

## Reverse order

Use the `reverse` prop to change the stacking order of avatars.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column fillWidth gap="24">
      <AvatarGroup
        avatars={[
          { value: "A" },
          { value: "B" },
          { value: "C" },
        ]}
        size="m"
        reverse={false}
      />
      <AvatarGroup
        avatars={[
          { value: "A" },
          { value: "B" },
          { value: "C" },
        ]}
        size="m"
        reverse={true}
      />
    </Column>
  }
  codes={[
    {
      code:
`<AvatarGroup
    size="m"
    avatars={[
      { value: "A" },
      { value: "B" },
      { value: "C" },
    ]}
/>

<AvatarGroup
    size="m"
    reverse
    avatars={[
      { value: "A" },
      { value: "B" },
      { value: "C" },
    ]}
/>`,
      language: "tsx",
      label: "Reverse"
    }
  ]}
/>
