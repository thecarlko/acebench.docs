`Media` handles responsive media content with support for lazy loading, aspect ratios, and intelligent rendering of videos or embeds. Use it for dynamic content blocks with consistent behavior.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Media
      src="/images/docs/once-ui/vibe-coding-light.jpg"
      alt="Preview"
      radius="xl"
      border="neutral-alpha-medium"
      aspectRatio="16/9"
    />
  }
  codes={[
    {
      code:
`<Media
    src="/image.jpg"
    alt="Preview"
    radius="xl"
    border="neutral-alpha-medium"
    aspectRatio="16/9"
/>`,
      language: "tsx",
      label: "Media"
    }
  ]}
/>

## YouTube

The component auto-detects and renders YouTube links, just pass a YouTube URL to the `src` prop.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  previewPadding="0"
  preview={
    <Media
      src="https://www.youtube.com/watch?v=TQT_FSdxbC8"
      aspectRatio="16/9"
    />
  }
  codes={[
    {
      code:
`<Media
    src="https://www.youtube.com/watch?v=TQT_FSdxbC8"
    aspectRatio="16/9"
/>`,
      language: "tsx",
      label: "YouTube"
    }
  ]}
/>
