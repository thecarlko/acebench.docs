The Kbd component is used to display keyboard keys and shortcuts with consistent styling. It's useful for showing keyboard interactions in documentation, tutorials, or application interfaces.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16" wrap>
      <Kbd>Tab</Kbd>
      <Kbd>Enter</Kbd>
      <Kbd>Shift</Kbd>
      <Kbd>Esc</Kbd>
    </Row>
  }
  codes={[
    {
      code:
`<Kbd>Tab</Kbd>
<Kbd>Enter</Kbd>
<Kbd>Shift</Kbd>
<Kbd>Esc</Kbd>`,
      language: "tsx",
      label: "Kbd"
    }
  ]}
/>

## Shortcuts

Use multiple Kbd components together to display keyboard shortcuts.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column gap="16" fillWidth textVariant="label-default-s" onBackground="neutral-weak">
      <Flex gap="8" vertical="center">
        <Kbd>Ctrl</Kbd>
        <Text>+</Text>
        <Kbd>C</Kbd>
        <Text>to copy</Text>
      </Flex>
      <Flex gap="8" vertical="center">
        <Kbd>Ctrl</Kbd>
        <Text>+</Text>
        <Kbd>V</Kbd>
        <Text>to paste</Text>
      </Flex>
      <Flex gap="8" vertical="center">
        <Kbd>Alt</Kbd>
        <Text>+</Text>
        <Kbd>Tab</Kbd>
        <Text>to switch windows</Text>
      </Flex>
    </Column>
  }
  codes={[
    {
      code:
`<Flex gap="8" vertical="center">
    <Kbd>Ctrl</Kbd>
    <Text>+</Text>
    <Kbd>C</Kbd>
    <Text>to copy</Text>
</Flex>`,
      language: "tsx",
      label: "Shortcuts"
    }
  ]}
/>

## Label

You can use either the `children` prop or the `label` prop to set the content of the Kbd component.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16">
      <Kbd label="Space"/>
      <Kbd>Enter</Kbd>
    </Row>
  }
  codes={[
    {
      code:
`<Kbd label="Space"/>
<Kbd>Enter</Kbd>`,
      language: "tsx",
      label: "Label"
    }
  ]}
/>

## Customization

The Kbd component inherits all Flex props, allowing for customization of its appearance.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16" vertical="center">
      <Kbd background="brand-medium" border="brand-medium" onBackground="brand-weak">Cmd</Kbd>
      <Kbd background="accent-medium" border="accent-medium" onBackground="accent-weak">Alt</Kbd>
      <Kbd paddingX="8" paddingY="4">Space</Kbd>
    </Row>
  }
  codes={[
    {
      code:
`<Kbd background="brand-medium" border="brand-medium" onBackground="brand-weak">Cmd</Kbd>
<Kbd background="accent-medium" border="accent-medium" onBackground="accent-weak">Alt</Kbd>
<Kbd paddingX="8" paddingY="4">Space</Kbd>`,
      language: "tsx",
      label: "Custom styling"
    }
  ]}
/>
