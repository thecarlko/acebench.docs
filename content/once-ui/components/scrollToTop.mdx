Use this utility to improve navigation for long-scrolling pages. It's customizable with positioning and visibility logic based on scroll offset.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <ScrollToTop offset={100} position="relative">
      <Button size="s" prefixIcon="arrowUp">Top</Button>
    </ScrollToTop>
  }
  codes={[
    {
      code:
`<ScrollToTop offset={100}>
    <Button size="s" prefixIcon="arrowUp">
      Top
    </Button>
</ScrollToTop>`,

      label: "ScrollToTop"
    }
  ]}
/>

## Offset

Define how far the user must scroll before the button becomes visible.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <ScrollToTop offset={400} position="relative">
      <IconButton icon="chevronUp"/>
    </ScrollToTop>
  }
  codes={[
    {
      code:
`<ScrollToTop offset={500}>
    <IconButton icon="chevronUp"/>
</ScrollToTop>`,
      language: "tsx",
      label: "Custom offset"
    }
  ]}
/>

## Custom style

The button can be customized using `Flex` props and additional `className`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <ScrollToTop position="relative" offset={600} className="my-custom-style">
      <Button size="s" variant="secondary" prefixIcon="arrowUp">Top</Button>
    </ScrollToTop>
  }
  codes={[
    {
      code:
`<ScrollToTop position="relative" className="my-custom-style">
  <Button size="s" variant="secondary" prefixIcon="chevronUp">
    Top
  </Button>
</ScrollToTop>`,
      language: "tsx",
      label: "Custom style"
    }
  ]}
/>
