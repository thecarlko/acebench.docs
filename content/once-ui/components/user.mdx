`User` is a compact component used to represent a user or persona. It combines an avatar, primary name, optional subline, and a status or role tag.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <User
      name="Lorant One"
      subline="Design Engineer"
      tagProps={{ label: "Admin", variant: "accent" }}
      avatarProps={{ src: "/images/avatar.jpg" }}
    />
  }
  codes={[
    {
      code:
`<User
    name="Lorant One"
    subline="Design Engineer"
    tagProps={{ label: "Admin", variant: "accent" }}
    avatarProps={{ src: "/images/avatar.jpg" }}
/>`,
      language: "tsx",
      label: "User"
    }
  ]}
/>

## Loading

Display skeletons with the `loading` prop to show a graceful placeholder before data is available.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <User loading name="Lorant One" subline="Design Engineer" />
  }
  codes={[
    {
      code:
`<User loading name="Lorant One" subline="Design Engineer" />`,
      language: "tsx",
      label: "Loading"
    }
  ]}
/>

## Custom content

You can override the default layout using `children`. This is useful for inline customization (e.g. badges, quick actions).

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <User avatarProps={{ value: "Q" }}>
      <Column paddingX="8">
        <Text variant="label-default-m">
          Quasar
        </Text>
        <Text variant="body-default-xs" onBackground="neutral-weak">
          Cyberforge
        </Text>
      </Column>
    </User>
  }
  codes={[
    {
      code:
`<User avatarProps={{ value: "Q" }}>
    <Column paddingX="8">
      <Text variant="label-default-m">
        Quasar
      </Text>
      <Text variant="body-default-xs" onBackground="neutral-weak">
        Cyberforge
      </Text>
    </Column>
</User>`,
      language: "tsx",
      label: "Custom content"
    }
  ]}
/>
