`Tooltip` is a non-interactive floating container used to communicate short hints or descriptions. It can be rendered manually or combined with a hover/focus trigger component. It is automatically hidden below the `m` breakpoint.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Tooltip label="Tooltip" />
  }
  codes={[
    {
      code:
`<Tooltip label="Tooltip" />`,
      language: "tsx",
      label: "Tooltip"
    }
  ]}
/>

## Icons

Add a leading or trailing icon to reinforce meaning.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Tooltip prefixIcon="info" label="Helpful tip" />
  }
  codes={[
    {
      code:
`<Tooltip prefixIcon="info" label="Helpful tip" />`,
      language: "tsx",
      label: "Icons"
    }
  ]}
/>

## Custom styles

You can override styles using `Flex` props, `style` or `className`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Tooltip prefixIcon="info" label="Helpful tip" background="brand-medium" border="brand-medium" padding="8" />
  }
  codes={[
    {
      code:
`<Tooltip prefixIcon="info" label="Helpful tip" background="brand-medium" border="brand-medium" padding="8" />`,
      language: "tsx",
      label: "Custom styles"
    }
  ]}
/>
