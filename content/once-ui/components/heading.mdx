The Heading component is used for displaying titles and section headings throughout your application. It provides consistent typography with various sizes, weights, and styles to establish visual hierarchy.

## Wrapping

Control how text wraps with the `wrap` prop. By default, the `wrap` prop is set to `balance`.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column gap="16" maxWidth={32}>
      <Heading>This text uses balance wrapping for optimal readability</Heading>
      <Heading wrap="normal">This text uses normal wrapping behavior</Heading>
    </Column>
  }
  codes={[
    {
      code:
`<Heading>
    This text uses balance wrapping for optimal readability
</Heading>
<Heading wrap="normal">
    This text uses normal wrapping behavior
</Heading>`,
      language: "tsx",
      label: "Wrapping"
    }
  ]}
/>
