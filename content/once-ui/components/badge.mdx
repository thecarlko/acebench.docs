A versatile component for displaying highlighted information, labels, or interactive elements in a pill-shaped container. Commonly used to draw attention to important information or provide interactive links.

## Link

Badges can be made interactive by adding the `href` prop, turning them into links.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16" horizontal="center">
      <Badge href="#" id="badge-4" title="Documentation" icon="pages" textVariant="label-default-s" />
    </Row>
  }
  codes={[
    {
      code:
`<Badge href="#" id="badge-4" title="Documentation" icon="pages" textVariant="label-default-s" />`,
      language: "tsx",
      label: "Interactive badges"
    }
  ]}
/>

## Arrow

Badges can be displayed without the arrow indicator by setting the `arrow` prop to `false`. If using multiple badges on a page, add custom `id` to each badge to ensure the arrow gets rendered.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16" horizontal="center">
      <Badge id="badge-5" title="With arrow"/>
      <Badge title="Without arrow" arrow={false} />
    </Row>
  }
  codes={[
    {
      code:
`<Badge id="badge-5" title="With arrow"/>
<Badge title="Without arrow" arrow={false} />`,
      language: "tsx",
      label: "Arrow"
    }
  ]}
/>
