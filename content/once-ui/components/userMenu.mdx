The `UserMenu` component renders a user avatar and name, and opens a dropdown for settings or account-level actions when clicked or focused. It is built on top of `DropdownWrapper` and `User`components.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <UserMenu
      name="Lorant One"
      subline="Design Engineer"
      placement="right-end"
      avatarProps={{ src: "/images/avatar.jpg" }}
      dropdown={
        <Column gap="4" padding="4" minWidth={10}>
          <ClientOption fillWidth hasPrefix={<Icon size="xs" onBackground="neutral-weak" name="settings" />} label="Settings" />
          <ClientOption fillWidth hasPrefix={<Icon size="xs" onBackground="neutral-weak" name="logout" />} label="Log out" />

## Loading state

You can show skeleton content while loading profile data by setting `loading={true}`.

<CodeBlock
  marginTop="16"
  marginBottom="24"

    <UserMenu loading />
  }

      code:
`<UserMenu loading />`,
      language: "tsx",
      label: "Loading"
    }
  ]}
/>
