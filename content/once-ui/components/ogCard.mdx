The `OgCard` component displays rich link previews for URLs by fetching and rendering Open Graph metadata. It's perfect for showcasing external links with preview images, titles, and descriptions.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <OgCard 
      url="https://once-ui.com"
    />
  }
  codes={[
    {
      language: "tsx",
      label: "OgCard",
      code:
`<OgCard 
    url="https://once-ui.com"
/>`,
    }
  ]}
/>

## Card styling

The `OgCard` extends the `Card` component, so you can use all `Card` props to customize its appearance.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <OgCard 
      url="https://once-ui.com/customize"
      background="brand-alpha-weak"
      border="brand-alpha-medium"
      shadow="l"
    />
  }
  codes={[
    {
      code:
`<OgCard 
    ogData={{
        title: "Styled Card",

        url: "https://example.com"

    background="brand-alpha-weak"
    border="brand-alpha-medium"
    shadow="l"
/>`,
      language: "tsx",
      label: "Card styling"

  ]}

## Notes

- Either `url` or `ogData` must be provided for the component to render
- The component automatically fetches favicons for the domain
- Images are proxied through an API endpoint to avoid CORS issues
- If both `url` and `ogData` are provided, `ogData` takes precedence
