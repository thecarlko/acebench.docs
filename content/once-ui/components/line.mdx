The Line component is used to create horizontal or vertical divider lines with consistent styling. It's useful for visually separating content sections in your application.

## Vertical line

Use the `vert` prop to create a vertical line instead of a horizontal one.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="32" height="80" center>
      <Text>Left side</Text>
      <Line vert height="80" />
      <Text>Right side</Text>
    </Row>
  }
  codes={[
    {
      code:
`<Row gap="32" height="80" center>
    <Text>Left side</Text>
    <Line vert height="80" />
    <Text>Right side</Text>
</Row>`,
      language: "tsx",
      label: "Vertical"
    }
  ]}
/>

## Custom styling

The Line component inherits all Flex props, allowing for customization of its appearance.

<CodeBlock
  marginTop="16"
  marginBottom="24"

      <Line background="brand-strong" />
      <Line background="accent-strong" />
      <Line background="danger-strong" />
