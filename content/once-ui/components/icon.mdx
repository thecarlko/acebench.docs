The Icon component is used to display vector icons throughout your application. It provides consistent styling with various sizes and colors to enhance visual communication.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16" horizontal="center">
      <Icon name="check" onBackground="info-medium" />
      <Icon name="danger" onBackground="danger-medium" />
      <Icon name="info" onBackground="brand-medium" />
      <Icon name="warning" onBackground="warning-medium" />
    </Row>

<Icon name="warning" onBackground="warning-medium"/>`,

## Accessibility

By default, icons are considered decorative and hidden from screen readers. Use the `decorative` prop to make icons accessible to screen readers when they convey meaning.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row gap="16">
      <Icon name="check" decorative={true} />
      <Icon name="warning" decorative={false} />
    </Row>
  }
  codes={[
    {
      code:
`// Hidden from screen readers (default)
<Icon name="check" decorative={true} />

// Accessible to screen readers
<Icon name="warning" decorative={false} />`,
      language: "tsx",
      label: "Accessibility"
    }
  ]}
/>

## Icon library

The Icon component uses icons from the [react-icons](https://react-icons.github.io/react-icons/) library. The available icons are defined in the icon library file. For more information about the icon library, see the [Icons](/once-ui/icons) page.
