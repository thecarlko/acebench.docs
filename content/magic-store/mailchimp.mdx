## Mailchimp account

Sign up for a [Mailchimp account](https://mailchimp.com) and create a new list if you don't have one.

## Embed form

Create a new [embed form](https://mailchimp.com/help/add-a-signup-form-to-your-website/) and copy the form URL to your Magic Store config.

<CodeBlock
    highlight="2"
    codes={[
  {
    code:
`const mailchimp = {
    action: 'https://url/subscribe/post?parameters',
};`, 
    language: "tsx",
    label: "app/resources/config.js"
  }
]} />

## Newsletter content

You can modify the headline and description in the `config.js` file.
